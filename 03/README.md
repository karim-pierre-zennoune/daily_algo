Wake-Up Kata — findCheapestProduct()
Écris une fonction findCheapestProduct en TS qui prend en paramètre un tableau de Product et retourne le produit le moins cher.
Un Product est un objet qui possède une propriété id, name et price

Tu peux tester ta fonction avec ce tableau :
const products = [
  { id: 1, name: "Pâtes bio", price: 2.5 },
  { id: 2, name: "Aspirateur", price: 89.9 },
  { id: 3, name: "Casque Bluetooth", price: 59.0 },
];
