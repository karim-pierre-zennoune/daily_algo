Exercice — Tester une fonction avec Vitest,
La fonction formatName(firstname, lastname) :
retourne une chaîne de caractères au format : "Firstname LASTNAME",
met la première lettre du prénom en majuscule,
met le nom de famille entièrement en majuscule,

---

Étape 1 — Écriture des tests,
Dans un premier temps, seuls les tests doivent être rédigés.
L’implémentation de la fonction ne doit pas être réalisée à ce stade.

Dans un fichier de test :

définis un bloc describe,
écris plusieurs cas de test avec it (des arguments différents),
utilise expect pour vérifier les résultats attendus,

---

Étape 2 — Implémentation,
Une fois les tests rédigés :

implémente la fonction formatName,
vérifie que l’ensemble des tests est valid
